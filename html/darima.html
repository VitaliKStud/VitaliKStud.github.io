class darima.MapDarima(*args, **kwargs)[source]

Is the main class for the Map step. This class inherit Darima class. Mapping every partition and calculating the coefficients. Uses R intergrated functions like auto.arima and ts. Also preparing a partition for calculations and convertering given partition into a timeseries. map_arima is the main-method within this class.
